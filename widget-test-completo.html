📋 Cómo Generar Métricas para el Dashboard

1. El widget del chat aparecerá en la esquina inferior derecha

Si no aparece, revisa la consola del navegador (F12) para ver errores

2. Haz clic en el ícono del chat para abrirlo
3. Prueba estas preguntas variadas para generar diferentes tipos de métricas:
"¿Qué es Python y para qué se usa?"
"Dame 5 consejos de marketing digital"
"¿Cómo diseñar un logo profesional?"
"Ideas innovadoras para mi negocio"
"Tendencias tecnológicas 2024"
"¿Cómo mejorar mi productividad?"

💡 Tip: Haz clic en cualquier pregunta para copiarla al portapapeles

4. Cada conversación generará automáticamente:
  • ✅ Registro de tokens consumidos - Para facturación
  • ✅ Datos del proveedor de IA usado - OpenAI, Google, DeepSeek
  • ✅ Categorización del tema - Para análisis de contenido
  • ✅ Métricas de rendimiento - Tiempo de respuesta, etc.
  • ✅ Eventos de analytics - Para dashboard administrativo
5. Después de 5-10 preguntas, ve al dashboard de analíticas:
  • 🔗 Panel de Usuario: http://161.132.45.210:5174/analytics
  • 🔗 Login: http://161.132.45.210:5174/login
💡

Tip para Mejores Resultados:

Haz preguntas en diferentes categorías (tecnología, marketing, diseño, negocios) para ver variedad en los "Temas Más Preguntados" y generar datos más ricos en el dashboard.

📊

Métricas que se Generarán:

Usuarios: Total de tokens usados, costo mensual, conversaciones por día
Súper Admins: Métricas de toda la plataforma, usuarios más activos, proveedores más usados, ingresos por planes
